형변환 함수(오라클 전용)
	숫자	-->	문자	 -->	날짜
		<--		<--

TO_CHAR()
		1.숫자를 문자로
		2.날짜를 문자로

TO_NUMBER()
		문자를 숫자로
TO_DATE()
		문자를 날짜로

CAST연산자(어디서든 사용 가능)
	CAST(값 AS 타입)

값의 타입
	숫자  : NUMBER(크기) -->크기 = 자릿수
		
	문자 : VARCHAR2(크기) --> 크기 = byte
	날자 : DATE - 9999년 까지 표현


NVL(값1,값2) : NULL값을 제어한다.
	값1이 만약 NULL 이라면 결과는 값2
	값1이 만약 NULL이 아니라면 결과는 값1

	NVL(10,'안녕') --> 10
	NVL(NULL,'안녕') --> 안녕

DECODE(값1, 값2, 값3, 값4) : 조건을 제시함수
	값1이 만약 값2랑 같다면 결과는 값3
	값1이 만약 값2랑 다르다면 결과는 값4

DECODE(값1,값2,DECODE(값3,값4,값5,값6),값7)
중첩 디코드 -> 순서대로 위치를 파악한다.
	값1이 값2라면 
		->값3이 값4라면 결과는 값5
		->값3이 값4가 아니라면 결과는 값6
	값1이 값2가 아니라면 결과는 값7

DECODE(값1,값2,값3,값4,값5,값6) = DECODE(값1,값2,값3,DECODE(값1,값4,값5,값6))              
	값1이 값2라면 결과는 값3
	값1이 값2가 아니라면 
		->값1이 값4라면 결과는 값5
		->값1이 값4가 아니라면 결과는 값6
값1로 두번 검사를 할경우 생략가능

CASE 표현식 : DECODE를  많이 중첩해 사용할경우 가독성이 너무 떨어진다. 조건마다 분기처리를 하고싶다 -> CASE사용
	CASE
		WHEN 조건1 THEN 값
		WHEN 조건2 THEN 값2
		WHEN 조건3 THEN 값3
		......
		ELSE 값n(생략가능)
	END
	조건1을 만족하면 결과는 값1
	조건1을 만족하지 않고 조건2를 만족하면 결과는 값2
	조건1, 조건2를 만족하지 않고 조건3을 만족하면 결과는 값3
	어떠한 조건도 만족하지 않으면 결과는 값n

**CASE 주의사항**
1.위에서부터 조건을 검색해 하나라도 만족하면 만족한 조건 밑의 조건은 모두 검사하지 않는다.
2.만약 검사하려는 조건의 값이 같고 =으로 비교한다면 생략하고 CASE 옆에 검사할 조건의 값을 적는다.

단일행 함수
	함수 사용결과가 각각의 행에 대해서 적용된다.
다중행 함수
	함수 사용 결과가 행의갯수랑 다르다.

집계함수
	SUM() : 총합
	AVG() : 평균
	COUNT() : 행갯수(데이터의 갯수)
	MIN() : 최솟값
	MAX() : 최댓값
**결과가 한개이므로 다른 컬럼과 같이 조회하지 못한다.
**NULL값은 항상 제외하고 계산한다.


GROUP BY 절

HAVING절
	다중행함수의 사용 결과를 바탕으로 행을 골라낼 때 사용한다.
